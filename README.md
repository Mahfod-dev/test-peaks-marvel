## Installation and Setup Instructions

#### Example:

Clone down this repository. You will need `node` and `npm` installed globally on
your machine.

Installation:

`npm install`

To Run Test Suite:

`npm test`

To Start Server:

`npm run dev`

To Visit App:

`localhost:5173 or localhost:3000`
